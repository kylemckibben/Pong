# Pong (a tribute)
Pong clone that started with the first chapter project in "Game Programming in C++: Creating 3D Games" by Sanjay Madhav.
The initial result has 3 walls and just a single player, with additional assignments to add a second player, and to add multiple balls. 
I am adding a second player, but not multiple balls, since I am instead choosing to stay true to the original with my clone.

## Current state
2 player support (without scoring).
The ball also does not change velocity on initial contact with the paddle, and it spawns in the same manner every time on game start.
There is no menu, to quit you either hit 'esc', click the 'x' on the window, alt+f4, etc.

## Upcoming features
- Menu to start the game
- Score display + game end on 11 points + winner display
- Release with an executable
- Single player vs NPC support (will make an initial attempt, but I don't want to burn too many cycles figuring it out right now since game AI is a section in the book)
